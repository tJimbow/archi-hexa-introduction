<h4>1️⃣ Le Domaine (Domain)</h4>

<ul>
  <li class="fragment"><strong>Indépendant</strong> de toute infrastructure</li>
  <li class="fragment">Logique métier pure (ex : calculer le montant d'une commande, appliquer une remise)</li>
  <li class="fragment">Règles métier (ex : une commande ne peut pas être validée si le stock est insuffisant, un client doit avoir une adresse)</li>
  <li class="fragment">Entités métier (ex : Client, Commande, Produit)</li>
</ul>
