<h4>2️⃣ Les Ports</h4>

<ul>
  <li class="fragment">Interfaces définissant les <strong>contrats</strong></li>
  <li class="fragment">Ports <strong>entrants</strong> (Driving / <strong>Primary</strong>) : API exposée par le domaine (<strong>API</strong> = Application Programming Interface)</li>
  <li class="fragment">Ports <strong>sortants</strong> (Driven / <strong>Secondary</strong>) : API dont le domaine a besoin (<strong>SPI</strong> = Service Provider Interface)</li>
</ul>
